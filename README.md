## DSDE Final Project 2025 by Yogurt Lover

Steps to Run the Project

1.Download all files from this GitHub repository.

2.Download the raw Traffy Fondue dataset from:
  https://drive.google.com/file/d/19QkF8i1my99gjbyHe7de_qZNwgrca6R5/view?usp=sharing

3.Run the .ipynb notebook to perform data cleansing, web scraping / API calls, and AI/ML processing.

4.Run app.py to launch the data visualization dashboard.
